This loader is a dumb wrapper around `dotenv`, it is useful to load envs from local files.

## Dependencies

<Tabs groupId='package-manager' persist items={['npm', 'pnpm', 'yarn']}>

```bash tab="npm"
npm install dotenv
```

```bash tab="pnpm"
pnpm add dotenv
```

```bash tab="yarn"
yarn add dotenv
```

</Tabs>

## Importing the loader

```ts title="env.config.ts"
import { config, adapters } from "fatima";
import dotenv from "dotenv";

export default config({
  load: {
    development: [adapters.dotenv.load(dotenv)],
  },
});
```
