## Dependencies

<Tabs groupId='package-manager' persist items={['npm', 'pnpm', 'yarn']}>

```bash tab="npm"
npm install -g vercel
```

```bash tab="pnpm"
pnpm i -g vercel
```

```bash tab="yarn"
yarn global add vercel
```

</Tabs>

## Authenticating your Vercel CLI

First, log in:

```bash
vercel login
```

Then, link your project:

```bash
vercel link
```

## Importing the loader

```ts title="env.config.ts"
import { config, adapters } from "fatima";
import { parse } from "dotenv";

export default config({
  load: {
    development: [adapters.vercel.load(parse)],
  },
});
```
